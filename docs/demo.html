Lotería Demo

This is a simple test to verify our webapp files are working correctly.

The main webapp is available at index.html

How to Use:

  1. Open index.html in a web browser
  2. Upload photos using "Upload Cards" with drag & drop
  3. Create at least 16 cards for full gameplay
  4. Generate printable tablas with "Generate Tabla"
  5. Use "Card Draw" mode for iPad presentation
  6. Export/Import decks for cross-device sync
  7. Install as PWA for offline use

Current Features:

  • 📸 Photo upload with drag & drop interface
  • 🖨️ Printable tabla generation (3:4 aspect ratio)
  • 🎴 Complete deck printing with cut guidelines
  • 📱 iPad-optimized Card Draw presentation mode
  • 🔄 Cross-device import/export functionality
  • 📶 Offline-first PWA with service worker caching
  • 💾 IndexedDB storage for unlimited capacity
  • 🎯 Touch-friendly interface for presentations
